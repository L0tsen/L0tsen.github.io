Varför Windows 12 troligen kommer vara dåligt

  • Vi såg nyligen när Windows 11 kom och var en total flop för Power Användare. Microsoft hadde tagit bort mer av kontrollpanelen och ersat det med barnleksaken de kallar inställningar.
  • Detta kommer bara att fortsätta vidare och vidare tills de har tagit bort kontrollpanelen helt och hållet. Vilket kommer vara dåligt för hemanvändaren och power Användare.
  • Microsoft vill att man ska byta till linux eller mac OS om de tar bort kontrollpanelen helt. Windows 8.1 var den sista vertionen av windows där man kunde göra alt i kontrollpanelen.
  • Men många gillade inte windows 8.1 pågrund av att de hadde tagit bort startmenyn och erast det med det bästa startprogramet för surfplattor.
  • Windows 8.1 var faktigst den snabbaste windows vertionen och med några program som Open shell så var det nog den bästa windws vertionen någonsin.
